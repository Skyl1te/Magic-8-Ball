# 🎱 Magic 8 Ball

A simple **Magic 8 Ball web application** built with **HTML, CSS, and JavaScript**.
Ask a question, click the button, and the ball will give you a random answer.

## 🚀 Features

* Random answers like the classic Magic 8 Ball
* Clean and simple UI
* Interactive button to generate answers
* Lightweight — no frameworks required

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**

```
magic-8-ball/

```

## ▶️ How to Run

1. Download or clone the repository:

```
git clone https://github.com/yourusername/magic-8-ball.git
```

2. Open the project folder.

3. Open **index.html** in your browser.

That's it! 🎉

## 💡 How It Works

1. The user enters a question.
2. When the **Ask** button is clicked:

   * JavaScript selects a random answer from an array.
   * The answer is displayed inside the ball.

Example of answers array:

```javascript
const answers = [
  "Yes",
  "No",
  "Maybe",
  "Ask again later"
];
```
